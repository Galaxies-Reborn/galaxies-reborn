# Pre-CU

Pre-Combat Upgrade era source sets.

No variant is published yet. When a Pre-CU source set exists, add its submodules
under `pre-cu/<variant>/`, add a manifest at `channels/pre-cu/<variant>.json`,
and flip `available` to `true` for the `pre-cu` flavor in `channels/index.json`.

Restoration research toward this era is tracked in
[`galaxies-reborn-pre-cu`](https://github.com/Galaxies-Reborn/galaxies-reborn-pre-cu).

Reborn Launcher lists this flavor but offers no variants until then, and
downloads nothing for it.
